e commerce

e commerce – is a web-based project that focuses on nostalgic Indonesian snacks and classic toys. The website was developed as an e-commerce platform with both user-facing and admin-side functionalities.

The user interface design was created collaboratively with a teammate, while I was responsible for implementing the interface into a functional website and developing the system features.

This website includes a shopping cart system, product management, and an admin dashboard. Users can browse products, add items to the cart, and manage their orders. On the admin side, the system provides full CRUD (Create, Read, Update, Delete) functionality for managing products, categories, and transaction data through a structured dashboard.

The project was built with a focus on responsive layout, usability, and clear information hierarchy, ensuring a smooth user experience across different devices.
